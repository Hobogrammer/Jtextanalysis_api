# JapaneseTextAnalyzer

Wrapper for the Yahoo Japanese text analysis api.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'japanese_text_analyzer'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install japanese_text_analyzer
